How it works :

- This quiz is all about knowing your partner quite well.

- It consists of 3 rounds namely 'Relationship Trivia', 'Convos', & 'Choices'.

- At start of each round you will get to know about the rules of that round.

- Don't worry, bae is always there to guide you.

- There are 3 modes namely 'Veteran', 'Intermediate', & 'Noob'.

- At the beginning of each round you will pe prompted to choose the mode.

- Each mode has it's own points system.

- At the end of 3 rounds your overall score will be calculated.

- No matter which mode you have chosen, questions of that round will always be the same.

- Veteran Mode : For each correct answer you get +5 and -3 if it is wrong.

- Intermediate Mode : For each correct answer you get +3 and -1 if it is wrong.

- Noob Mode : For each correct answer you get +2 and 0 if it is wrong.

- Your cumulative score decides in which winning/losing category you fall.

- The categories are as follows, rewards/risks will be disclosed later (better the category better the reward).

- 3 Rounds, 7 questions each, when played each in Veteran Mode, leads to Max score as +105 and Min score as -63

- Categories :

70 - 105 : Tumhara Koi Jawaab Nahi

50 - 70 : Dil Khush

40-50 : Kaam Chala Lunga

20 - 40 : Kya hi tum

Below 20 : Bhool jao Reward and you owe me something instead!
